# CS2 Server Management System

A comprehensive server management system for Counter-Strike 2 servers.

## Features

- Full RCON support
- Plugin system
- Metrics collection
- State management
- Error handling
- Logging system
- API client
- Configuration management

## Requirements

- Python 3.8+
- CS2 Dedicated Server
- Required Python packages (see requirements.txt)
